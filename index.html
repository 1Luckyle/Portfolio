Mes Compétences

HTML, CSS & JavaScript

HTML, CSS et JavaScript forment la base du développement web. HTML structure la page, CSS la stylise et JavaScript gère l'interactivité côté client.

Proxmox

Proxmox est une plateforme open source de virtualisation et de gestion de conteneurs. Elle permet de déployer et d’administrer facilement des environnements virtualisés.

Conteneurisation

La conteneurisation permet d’isoler et de déployer des applications dans des environnements reproductibles à l’aide d’outils comme Docker ou Kubernetes.

SQL

SQL est un langage de programmation utilisé pour gérer et manipuler des bases de données relationnelles. Il est essentiel pour la récupération et la mise à jour des données dans les applications.

C & C++

Le C et le C++ sont des langages à bas niveau offrant un contrôle précis de la mémoire et des performances élevées. Ils sont utilisés pour la création de systèmes, de jeux et de logiciels exigeants.

Cypress

Cypress est un framework de test end-to-end moderne et rapide pour le web, qui facilite la création, l’exécution et la maintenance de tests automatisés.
