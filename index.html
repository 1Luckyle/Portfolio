Mes Compétences

HMTL5

HTML5 est la dernière version du langage de balisage standard utilisé pour créer des pages web. Il offre de nouvelles fonctionnalités pour améliorer la sémantique, l'accessibilité et la performance des sites web.

CSS3

CSS3 est utilisé pour décrire la présentation des documents HTML. Il permet de créer des designs réactifs, des animations et des transitions, améliorant ainsi l'expérience utilisateur.

Java

Java est un langage de programmation orienté objet utilisé pour développer des applications web, mobiles et de bureau robustes et sécurisées.

SQL

SQL est un langage de programmation utilisé pour gérer et manipuler des bases de données relationnelles. Il est essentiel pour la récupération et la mise à jour des données dans les applications.

Assembleur 68000

L'assembleur 68000 est un langage de bas niveau utilisé pour programmer le processeur Motorola 68000. Il est utilisé dans les systèmes embarqués et pour la programmation de matériel spécifique.

Script R

R est un langage de programmation et un environnement logiciel libre pour les statistiques et l'analyse des données. Il est largement utilisé dans les domaines de la data science et de la recherche académique.
